# 🌍 WorldWise — React Application

## 📌 About the Project

### WorldWise is a single-page application built with React that allows users to track places they have visited around the world. Users can add cities on the map, store notes about their trips, and explore their travel history in an interactive interface.

### This project was developed as part of the React course by Jonas Schmedtmann, focusing on modern React development practices, component architecture, routing, state management, and working with APIs.

## 🚀 Features

### 🗺 Interactive map with city markers

### ➕ Add new visited cities by clicking on the map

### 📝 Save notes and travel dates

### 📍 View detailed city information

### 🧭 Navigation between pages using React Router

### 💾 Persistent data via fake REST API (JSON Server)

### ⚡ Fast development environment powered by Vite

### 🎨 Modular and responsive UI

## 🛠 Technologies Used

### React (Hooks, Functional Components)

### React Router

### Context API for global state management

### Custom Hooks

### CSS Modules

### Leaflet / React-Leaflet for map rendering

### JSON Server for mock backend

### Vite as build tool
